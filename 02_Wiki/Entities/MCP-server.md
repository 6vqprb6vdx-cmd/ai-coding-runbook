## 一句话定义

通过 MCP 协议接入 Claude Code 的外部 tool 服务器，提供资源和工具

## 关键属性

- 通过 Model Context Protocol 将外部 DB / API / 服务以 tools 形式接入 Claude Code，无需写自定义 tool 代码 [[mcp--agent-sdk]] [[mcp-integration--skill]]
- 四种 transport：**stdio**（本地子进程，JSON-RPC over stdin/stdout）、**SSE**（hosted + OAuth）、**HTTP**（REST，token auth via headers）、**WebSocket**（持久双向，仅 WSS） [[server-types--plugin-dev]] [[mcp--agent-sdk]]
- 配置可放 `query()` 的 `mcpServers` 选项，或项目根的 `.mcp.json`；plugin 内推荐独立 `.mcp.json`，inline `plugin.json.mcpServers` 适合单 server [[mcp--agent-sdk]] [[mcp-integration--skill]]
- Tool 命名规范 `mcp__<server>__<tool>`；plugin 内额外加 plugin 前缀变成 `mcp__plugin_<plugin>_<server>__<tool>`，可用 `/mcp` 列出全名 [[mcp--agent-sdk]] [[tool-usage--mcp-integration]]
- Tool 必须在 `allowedTools` 显式 allow，wildcard `mcp__github__*` 可一次放过整个 server；`acceptEdits` 模式不会 auto-approve MCP tool，只有 `bypassPermissions` 会 [[mcp--agent-sdk]]
- stdio server 通过 `env` 字段传凭证；SSE/HTTP 通过 `headers`（如 `Authorization: Bearer ${TOKEN}`）；OAuth 2.1 由 Claude Code 自动跑浏览器 flow，token 加密存储且 plugin 不可见 [[authentication--mcp-integration]] [[mcp--agent-sdk]]
- 支持 `${CLAUDE_PLUGIN_ROOT}` / `${CLAUDE_PLUGIN_DATA}` / `${VAR}` 环境变量替换，便于 plugin 引用打包的 binary 和 config [[plugins-reference]] [[stdio-server-mcp-example--plugin-dev]]
- 启动时 SDK 发出 `system` init message，`mcp_servers[].status` 可用来检测连接失败（默认 60s timeout） [[mcp--agent-sdk]]
- `dynamic headers` via `headersHelper` script — 适用于短期 token / HMAC 签名 / 时间相关 auth [[authentication--mcp-integration]]
- **Tool search** 默认开启：MCP tool 定义按需加载，避免 context window 被几十个 tool schema 撑爆 [[mcp--agent-sdk]]
- Best practice：tool 命名 `{service}_{action}_{resource}` snake_case；server 命名 Python `{service}_mcp` / Node `{service}-mcp-server`；list tool 必带 pagination + `has_more`/`next_offset` [[mcp_best_practices]]
- 推荐 stack（mcp-builder skill）：TypeScript + Streamable HTTP（remote）/ stdio（local）；**SSE 已 deprecated**，新项目用 streamable HTTP [[mcp-builder-skill]] [[mcp_best_practices]]
- Anthropic 不审计任何第三方 MCP server — 只用 trusted 源或自己写；首次连新 MCP server 触发 trust prompt（headless `-p` 模式禁用） [[security]]
- `npx @anthropic-ai/sandbox-runtime <command>` 可把 MCP server 跑在 sandbox 里增加隔离 [[sandboxing]] [[secure-deployment]]
