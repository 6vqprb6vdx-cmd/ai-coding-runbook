LlamaIndex <> Claude cookbook directory. LlamaIndex is a data framework for LLM-based applications that benefit from context augmentation. The cookbook ships notebooks for building LLM applications with Anthropic models and LlamaIndex.

**Notebooks:**

1. `Basic_RAG_With_LlamaIndex.ipynb` — building RAG pipelines.
2. `Router_Query_Engine.ipynb` — using `RouterQueryEngine` to route user queries to different indices.
3. `SubQuestion_Query_Engine.ipynb` — using `SubQuestionQueryEngine` to answer complex queries spanning multiple documents.
4. `ReAct_Agent.ipynb` — using `ReActAgent` with Tools and QueryEngine Tools.
5. `Multi_Document_Agents.ipynb` — efficient RAG pipeline for a large number of documents.
6. `Multi_Modal.ipynb` — multimodal applications with LlamaIndex.
