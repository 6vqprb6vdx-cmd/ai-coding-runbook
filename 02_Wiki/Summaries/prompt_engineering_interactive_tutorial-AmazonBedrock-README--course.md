Bedrock Edition of Anthropic's Prompt Engineering Interactive Tutorial.

**Outcomes.** Same as the main 1P version: master basic prompt structure, recognize common failure modes and 80/20 fixes, understand Claude's strengths/weaknesses, build strong prompts from scratch.

**Structure.** 9 chapters with exercises plus an appendix of advanced methods. Each lesson has an "Example Playground" area. The 1P answer key (Google Sheets) applies — solutions are the same even though it is structured for 1P API requests.

Note: tutorial defaults to Claude 3 Haiku for cost reasons.

**Table of contents:**

- **Beginner** — Chapter 1: Basic Prompt Structure; Chapter 2: Being Clear and Direct; Chapter 3: Assigning Roles.
- **Intermediate** — Chapter 4: Separating Data from Instructions; Chapter 5: Formatting Output & Speaking for Claude; Chapter 6: Precognition (Thinking Step by Step); Chapter 7: Using Examples.
- **Advanced** — Chapter 8: Avoiding Hallucinations; Chapter 9: Building Complex Prompts (Industry Use Cases) including Chatbot, Legal Services, Financial Services exercise, Coding exercise, Congratulations & Next Steps.
- **Appendix: Beyond Standard Prompting** — Chaining Prompts, Tool Use, Empirical Performance Evaluations, Search & Retrieval. (The Bedrock edition appendix adds "Empirical Performance Evaluations" relative to the 1P edition.)

The substantive lesson content is identical to the 1P edition — only the underlying API surface (Bedrock instead of direct Claude API) changes.
