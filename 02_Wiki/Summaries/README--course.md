Top-level README for Anthropic's educational courses repository. Currently contains five courses, suggested order:

1. **Anthropic API fundamentals** (`anthropic_api_fundamentals/`) — essentials of working with the Claude SDK: API key, model parameters, multimodal prompts, streaming responses, etc.
2. **Prompt engineering interactive tutorial** (`prompt_engineering_interactive_tutorial/`) — comprehensive step-by-step guide to key prompting techniques. Also has an AWS Workshop version linked.
3. **Real world prompting** (`real_world_prompting/`) — incorporating prompting techniques into complex, real-world prompts. Google Vertex version linked.
4. **Prompt evaluations** (`prompt_evaluations/`) — writing production prompt evaluations to measure prompt quality.
5. **Tool use** (`tool_use/`) — implementing tool use successfully with Claude.

Note: courses often default to the lowest-cost model (Claude 3 Haiku) to keep API costs down for students; learners can swap in other models.
