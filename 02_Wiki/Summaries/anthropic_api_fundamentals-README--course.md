A series of notebook tutorials covering the essentials of working with Claude models and the Anthropic SDK.

**Lessons:**

- `01_getting_started.ipynb` — Getting an API key and making simple requests.
- `02_messages_format.ipynb` — Working with the messages format.
- `03_models.ipynb` — Comparing capabilities and performance of the Claude model family.
- `04_parameters.ipynb` — Understanding model parameters.
- `05_Streaming.ipynb` — Working with streaming responses.
- `06_vision.ipynb` — Vision prompting.
