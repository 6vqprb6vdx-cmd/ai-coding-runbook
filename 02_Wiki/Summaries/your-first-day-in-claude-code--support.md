Goal: In about 15 minutes, you will install Claude Code, sign in, and complete your first AI-assisted change. This guide covers only what is needed on day one; links to deeper material appear at the end.

Pick the command that matches your machine and paste it into a terminal.

| Your setup | Command | | --- | --- | | macOS / Linux / WSL | `curl -fsSL <https://claude.ai/install.sh> | | Windows (PowerShell) | `irm <https://claude.ai/install.ps1> | | Homebrew | `brew install --cask claude-code` | | npm (Node 18+) | `npm install -g @anthropic-ai/claude-code` |

Covers: Step 1: Install Claude Code (2 minutes); Step 2: Sign in (2 minutes); Step 3: Open a project (1 minute); Step 4: Understand the one safety rule (1 minute); Step 5: Five things to try first; 1. Get oriented; 2. Find something; 3. Make a safe edit.
