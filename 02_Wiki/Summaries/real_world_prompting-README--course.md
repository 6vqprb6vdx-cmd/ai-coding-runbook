Anthropic's comprehensive Real World Prompting tutorial. Designed for experienced developers who have completed the Prompt Engineering Interactive Tutorial. Five lessons covering how to incorporate key prompting techniques into complex real-world prompts.

**Table of contents.**

- `01_prompting_recap.ipynb` — Prompting recap.
- `02_medical_prompt.ipynb` — Medical prompt walkthrough.
- `03_prompt_engineering.ipynb` — Prompt engineering process.
- `04_call_summarizer.ipynb` — Call-summarizing prompt walkthrough.
- `05_customer_support_ai.ipynb` — Customer support bot prompt walkthrough.

Lessons build on each other; recommended to start at lesson 01.
