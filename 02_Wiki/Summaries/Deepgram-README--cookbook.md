Deepgram <> Claude cookbook directory. Deepgram is a foundational AI company providing speech-to-text, text-to-speech, text-to-text, and language-intelligence capabilities to make data readable and actionable by humans or machines.

**Notebook:** `prerecorded_audio.ipynb` transcribes pre-recorded audio with Deepgram (intended to feed transcripts into Claude downstream).

**Resources.** API Playground, Starter Apps, Python/Node/.NET/Go SDKs, official documentation and blog. Community: Deepgram GitHub Discussions and Discord.
