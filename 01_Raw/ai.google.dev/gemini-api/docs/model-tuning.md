# الضبط الدقيق باستخدام Gemini API

مع إيقاف Gemini 1.5 Flash-001 نهائيًا في مايو 2025، لم يعُد لدينا
نموذج يتيح الضبط الدقيق في Gemini API أو AI Studio، ولكنّه
متاح في [Gemini Enterprise Agent Platform](https://cloud.google.com/gemini-enterprise-agent-platform/models/gemini-use-supervised-tuning?hl=ar).

ليس لدينا خطط فورية لإتاحة ميزة الضبط الدقيق، ولكن يسرّنا [معرفة
رأيك في منتدى المطوّرين](https://discuss.ai.google.dev/c/gemini-api/4?hl=ar)
إذا كانت هذه الميزة مهمة لحالة الاستخدام الخاصة بك.
