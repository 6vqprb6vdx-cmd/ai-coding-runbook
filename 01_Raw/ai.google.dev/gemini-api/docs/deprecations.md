# Gemini 지원 중단

이 페이지에는 Gemini API의 [안정화 (GA)](https://ai.google.dev/gemini-api/docs/models?hl=ko#stable) 및 [미리보기](https://ai.google.dev/gemini-api/docs/models?hl=ko#preview) 모델의 알려진 지원 중단 일정이 나와 있습니다. '**지원 중단**'은 더 이상 모델을 지원하지 않으며 가까운 시일 내에 '**종료**'될 것이라는 발표입니다. 모델이 '**종료**'되면 완전히 꺼지고 엔드포인트를 더 이상 사용할 수 없습니다.

지원 중단 공지는 [출시 노트](https://ai.google.dev/gemini-api/docs/changelog?hl=ko) 페이지에 게시되며, 공지된 가장 빠른 종료 날짜는 이 페이지에서 추적됩니다.
이미 종료된 모델은 회색 배경으로 표시됩니다.
