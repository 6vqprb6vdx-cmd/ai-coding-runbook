# Google AI Studio 和 Gemini API 的可用區域

如果您在嘗試存取 [Google AI Studio](https://aistudio.google.com/?hl=zh-tw) 時看到這個頁面，可能是下列其中一個原因所致：

- **區域限制**：Google AI Studio 尚未在你的所在區域推出。如需支援的區域清單，請參閱下方章節。
- **年齡規定**：你未滿 18 歲，不符合存取這項服務的最低年齡規定。
- **帳戶驗證**：你可能可以存取服務，但尚未在 Google 帳戶中[驗證年齡](https://support.google.com/accounts/answer/10071085?hl=zh-tw)。

如要進一步瞭解相關規定，請參閱《[服務條款](https://ai.google.dev/gemini-api/terms?hl=zh-tw)》。
