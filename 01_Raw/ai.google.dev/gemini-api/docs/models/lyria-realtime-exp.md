# Lyria RealTime 試験運用版

高忠実度の音楽合成のための試験的なエンジン。オーディオ コンテンツの生成と変換のための革新的な機能を提供します。Lyria RealTime は、AI を活用した作曲、独自のインストゥルメンタルの生成、メロディーとリズムを細かく制御する必要があるボーカルなしのクリエイティブなオーディオ ワークフローに最適です。
