## モデル バージョン名のパターン

Gemini モデルは、 *安定版* 、 *プレビュー版* 、 *最新版* 、または
*試験運用版* のいずれかのバージョンで利用できます。

### 安定版

特定の安定版モデルを参照します。通常、安定版モデルは変更されません。ほとんどの本番環境アプリでは、特定の安定版モデルを使用する必要があります。

例: `gemini-2.5-flash`。

### プレビュー

本番環境で使用できるプレビュー モデルを参照します。通常、プレビュー モデルでは課金が有効になっており、レート上限が厳しくなる可能性があります。また、少なくとも 2 週間前に通知されたうえで非推奨となります。

例: `gemini-2.5-flash-preview-09-2025`。

### 最新

特定のモデル バリエーションの最新リリースを参照します。これは、安定版、プレビュー版、試験運用版のリリースにすることができます。このエイリアスは、特定のモデル バリエーションの新作ごとにホットスワップされます。最新版のバージョンが変更される **2 週間前** に、メールで通知が届きます。

例: `gemini-flash-latest`。

### 試験運用版

試験運用版モデルを参照します。通常、本番環境での使用には適しておらず、レート上限が厳しくなります。Google は、フィードバックを収集し、最新のアップデートをデベロッパーに迅速に提供するために、試験運用版モデルをリリースしています。

試験運用版モデルは安定しておらず、モデル エンドポイントの可用性は変更される可能性があります。
